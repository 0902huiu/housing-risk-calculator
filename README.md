# 청년 주거 위험 조정 비용 비교 계산기

전세·반전세·월세의 총비용에 **기회비용**과 **전세사기 기대손실**(사고 확률 × 보증금 × 미회수율)을 더해, 눈에 보이지 않던 위험을 '원 단위 비용'으로 보여주는 웹 계산기입니다.

> "싸 보이는 선택이 반드시 최적의 선택은 아니다" — 이 명제를 데이터로 증명하는 것이 이 프로젝트의 목표입니다.

## 주요 기능

- 전세·반전세·월세의 **위험 조정 실질 총비용** 실시간 비교
- **전세가율 자동 계산**: 매매가 입력 시 자동 산출, 매매가를 모르면 공시가격 × 126%(HUG 기준)로 추정
- 전세가율 구간별 **사고 확률 차등 적용** (HUG 통계 기반: 70% 미만 1% ~ 90% 초과 12%)
- **전세보증보험(HUG) 토글**: 보험료 vs 기대손실의 경제성 비교, 전세가율 90% 초과 시 가입 불가 경고
- 표면 비용과 실질 비용의 **순위 역전 감지**
- 정량화할 수 없는 위험(서류 위조·권리관계형 사기)을 위한 계약 전 체크리스트

## 계산 모델

```
실질 총비용 = 직접 지출(월세·대출이자·보험료) + 기회비용 + 기대손실
기대손실 = 사고 확률(전세가율 구간별) × 보증금 × (1 − 회수율 40%)
```

## 데이터 출처

- 국회 국토교통위원회·HUG, 전세보증금반환보증 사고 현황 (2020~2025)
- 국토교통부, 전세사기 실태조사 (2025.6)
- KDI FOCUS, 「전세보증금 반환보증제도 개선방안」 (2023.9)
- 국토교통부 실거래가 공개시스템 — 실매물 검증 (인천 미추홀구·서울 대치동)
